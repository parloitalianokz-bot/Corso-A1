🛠️ Come Aggiungere o Modificare gli Studenti?

Poiché il sito è ospitato gratuitamente su GitHub Pages, il "database" si trova all'interno del file index.html (linee 191-197). Per iscrivere un nuovo studente o sbloccare livelli:

  1. Apri il file index.html sul tuo computer o direttamente su GitHub.
  2. Trova l'oggetto database all'inizio dello script.
  3. Aggiungi una nuova riga con il nome utente in minuscolo, la password convertita in Base64 (usa lo strumento qui sotto) e il numero massimo di unità accessibili (maxUnit).
  4. Salva il file e fai il commit/push su GitHub per rendere le modifiche attive!
// Esempio nel codice di index.html: "nomeutente": { pass: "codice_base64", maxUnit: 3 }
